# Screen-Saver
Screen Saver in ArmSim using Embest Board Plugin

Steps for executing this code:

    1) install armsim simulator
    2) Go to File-->Preferences->Plugins and select "Embest Board Plugin"
    3) Run the code
    4) Go to view-->PLuginsUI
